## How to run:
1. Initialize and run mysql server using [redacted].sql 
2. Make a Virtual Environment
```
    python -m venv .venv
```
3. Activate Virtual Environment
On Powershell
```
    .\.venv\Scripts\activate.ps1
```
On CMD
```
    .\.venv\Scripts\activate.bat
```
On Unix (untested)
```
    .\.venv\Scripts\activate
```
4. Run Python File
```
    python main.py
```
